🔄 EITHER

End the sentence wITH the same lettER — build sentences where each word starts and ends with the same letter

How to Play

EITHER is a sentence-building game with a creative twist. Form sentences using words that start and end with the same letter. Then the next sentence must start with the second-to-last letter of the previous sentence's last word.

Rules

  • Each word in your sentence should ideally start and end with the same letter.
  • The next sentence starts with the second-to-last (penultimate) letter of the previous sentence's last word.
  • Avoid proper nouns and the conjunction "and".
  • Build simple, grammatically correct sentences.

📖 Understanding the Pattern

Words that start and end with the same letter:

That Did Enjoyable Noon Ships Willow Radar Level

Example sentence chain:

1 That dad did good.
↓ penultimate letter of "good" = O
2 Olive oil offers overall options.
